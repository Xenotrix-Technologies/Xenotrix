Xenotune — AI Music Generation Platform

Xenotune is an AI-powered music generation system developed by Xenotrix Technologies. It creates infinite, mood-based soundscapes using deep learning and music theory. The platform is designed for focus, relaxation, and sleep, generating unique melodies every time.

Built with FastAPI as the backend and Flutter as the frontend, Xenotune blends advanced AI models with real-time audio synthesis. It uses music21 for composition, FluidSynth for MIDI rendering, and FFmpeg for smooth MP3 conversion.

“Xenotune redefines how AI interacts with music — generating emotion-driven, endless compositions that evolve as you listen.”

Backend & AI Integration

The app seamlessly integrates AI melody generation with real-time sound control through Flutter. It supports full playback control — play, pause, resume, and stop — along with smooth fade-in/out transitions and background blending.

Xenotune also connects with Firebase for secure file storage and metadata management, maintaining mood-specific data like tempo, instruments, and emotion mapping for each generated piece.
